# EasyPercent - Percentage Calculator

Simple, fast percentage calculator with instant updates (no submit buttons).

## Features

- What is X% of Y
- X is what percent of Y
- Increase / Decrease (absolute + %)
- Add / Subtract percentage from a base value
- Instant calculations while typing

## Run locally

```bash
cd percent-calculator-live
python3 -m http.server 8787
```

Open: http://localhost:8787

## Deploy

This is a static HTML app (`index.html`), so it can be deployed to:

- GitHub Pages
- Netlify
- Vercel (static)
- Any basic web server
